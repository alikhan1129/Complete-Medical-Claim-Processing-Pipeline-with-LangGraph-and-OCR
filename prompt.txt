# 🧠 LANGGRAPH FLOW (STRICT)

START
  ↓
Segregator Agent (Groq LLM classifier)
  ↓

   ├── ID Agent
   ├── Discharge Summary Agent
   ├── Itemized Bill Agent
  ↓
Aggregator
  ↓
END

IMPORTANT:
- Segregator runs FIRST
- Only 3 extraction agents exist
- Agents must NOT process full PDF
- They only receive relevant page text

# 🧩 SEGREGATOR AGENT (CRITICAL)

- Input: full PDF (page-wise text extracted using pdfplumber)
- Use Groq LLM to classify each page into:

  claim_forms
  cheque_or_bank_details
  identity_document
  itemized_bill
  discharge_summary
  prescription
  investigation_report
  cash_receipt
  other

OUTPUT MUST BE STRICT JSON ONLY:

{
  "page_map": {
    "1": ["identity_document"],
    "2": ["itemized_bill"]
  }
}

RULES:
- No explanations
- No markdown
- No extra text
- Only valid JSON
- Multiple labels allowed per page

---

# 🤖 EXTRACTION AGENTS (3 ONLY)

Each agent receives ONLY relevant page text.

---

## 1. ID Agent
Extract:
- patient_name
- dob
- id_number
- policy_number

Return STRICT JSON only.

---

## 2. Discharge Summary Agent
Extract:
- diagnosis
- admit_date
- discharge_date
- physician_name

Return STRICT JSON only.

---

## 3. Itemized Bill Agent
Extract:
- items: list of {name, cost}
- total_amount (must calculate correctly)

Return STRICT JSON only.

---

# 🔗 AGGREGATOR NODE

  "bill": {...}
